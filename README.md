## TheTVDB 

This library provides access to TheTVDB API.

Documentation can be found [here](https://hexdocs.pm/thetvdb/TheTVDB.html).

## Installation

```elixir
def deps do
  [{:thetvdb, "~> 1.0.0"}]
end
```

This library runs as an application, so ensure it's added to `mix.exs`:

```elixir
def applications do
  [applications: [:thetvdb]]
end
```
